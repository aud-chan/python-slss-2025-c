# Styling Characters

We typically use **bold** and *italics* to style characters when we type something.
You can do both ***bold and italics***
We can also use underscores to __bold__ and _italicize_ words

bold - two asterisks 
italics- one asterisk 
both bold and italics- three asterisks

If you want to use stars, use only stars. The same is true for underscores. In other words, `be consistent`

# Links 
Markdown is a language that translates from plain text to another language called `HTML`. `HTML` is the language that _web pages_ are built with. 

We can use links in our notes to other pages by using square brackets. `[]` 

[ChatGPT](https://chat.openai.com)
[Google](https://google.com)
